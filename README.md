# Bound to Books

Bound to Books is a premium book resale and donation platform designed for book lovers. Our mission is to ensure that every book finds a new soul to touch, connecting pre-owned stories with eager readers.

## 📚 Features

- **Marketplace**: Browse a curated selection of pre-loved books across various categories including Fiction, Non-Fiction, Classics, and more.
- **Sell/Donate**: Easily list your own books for sale or donation. Our team handles the logistics once a request is accepted.
- **User Dashboard**: Track your orders, monitor your selling/donation requests, and manage your profile.
- **Admin Console**: Robust management tools for administrators to approve requests, manage inventory, and track platform analytics.
- **Eco-Friendly**: We prioritize sustainability by recirculating books and using eco-friendly packaging.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (Node Package Manager)

### Local Development

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd BoundToBooks/frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Access the application**:
   Open your browser and navigate to `http://localhost:3000` (or the port specified in your terminal).

## 🛠️ Technology Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Vanilla CSS (Modern HSL-based design system)
- **Icons**: Lucide React
- **State Management**: React Context API
